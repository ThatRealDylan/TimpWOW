Getting Started

TimpWOW is a simple yet powerful Python-based package manager. Follow these steps to get started:

Installation

  1. Download TimpWOW from the Downloads page
  2. Extract the archive to your desired location
  3. Navigate to the directory in your terminal
  4. Run: python terminal.py

Commands

Command Description
help Shows all available commands
pkg install <name> Install a package
pkg remove <name> Remove an installed package
pkg list List installed and available packages
pkg info <name> Show package information
servers list Fetch available packages from GitHub
servers fetch <name> Get detailed package info
servers test Test connection to GitHub
update Check for updates
license Show MIT License

Package Management

Installing Packages

To install a package, use the following command:

pkg install package-name

The package manager will:

  1. Download the .timp file from GitHub
  2. Extract it to the pkg/extracted/ directory
  3. Save the original file in pkg/

Package Structure

.timp files are ZIP archives with the following structure:

project/ - Your project files plugins/ - TimpWOW plugins props.json - Package metadata

Creating Your Own Packages

Step 1: Create Directory Structure

Create a folder with your package files:

my-package/ ├── project/ │ └── (your project files) ├── plugins/ │ └── (your plugins) └── props.json

Step 2: Add Metadata

Create a props.json file with package information:

{ "name": "my-package", "version": "1.0.0", "author": "Your Name", "description": "A great package", "license": "MIT" }

Step 3: Create .timp File

Compress your folder as a ZIP file and rename it to .timp:

  1. Select all files in your package folder
  2. Compress to ZIP
  3. Rename the ZIP file to my-package.timp

Step 4: Upload to GitHub

Add your .timp file to the root of your GitHub repository. Users can now install it with:

pkg install my-package

FAQ

Where are my packages stored?

Installed packages are stored in two locations:

  • pkg/ - Original .timp files
  • pkg/extracted/ - Extracted package contents

How do I update a package?

Simply reinstall the package using pkg install package-name. The old files will be overwritten.

Can I create private packages?

Yes! Use a private GitHub repository and configure TimpWOW to use your repository URL.

What if I need help?

Visit the GitHub repository to report issues or ask questions.
